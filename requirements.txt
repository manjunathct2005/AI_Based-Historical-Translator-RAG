# IMPORTANT (Streamlit Community Cloud / limited-RAM deploys):
# Packages are ordered lightest-first and version-pinned (not just >=) so
# pip's resolver doesn't spend the build budget backtracking, and so a
# failure on a heavy package (torch/faiss/easyocr) doesn't silently skip
# installing things listed after it, like SQLAlchemy -- which is exactly
# what causes "ModuleNotFoundError: no module named X" deep inside app
# imports that look unrelated to the real failing package. Check the
# Streamlit Cloud build logs (Manage app -> logs) to see which line
# actually failed if you still hit an error after this fix.
#
# ChromaDB is intentionally NOT included by default: it pulls a very large
# dependency tree (onnxruntime, duckdb, kubernetes client, etc.) that
# frequently fails to build on free-tier cloud runners. FAISS (included
# below) is the default vector backend and needs none of that. If you want
# Chroma, install it yourself: pip install chromadb>=0.5.0
# ============================================================================
